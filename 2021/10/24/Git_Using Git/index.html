注：git reflog命令可以显示所有版本，git log命令只能显示当前所处版本之后的版本。

reset 3个命令参数的对比：

  • --soft：仅仅在本地库移动 HEAD 指针。

  • --mixed：在本地库移动 HEAD 指针，重置暂存区。

  • --hard：在本地库移动 HEAD 指针，重置暂存区，重置工作区。
